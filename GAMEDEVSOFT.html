Для Геймдизайнеров

Любой геймдизанер должен писать документацию (любой удобный текстовый редактор) для того, чтобы программистам, модельерам было проще понимать устройство игры, а также: Сетинг, необходимость каких либо механик, и атмосферу.

Для Программистов

Программисты должны использовать редакторы кода для написания команд, физики, движка, некоторые пишит ИИ, а также внедряют контент в игру.

Для
Модельеров

Модельеры должны при помощи программ для 3D моделмрования создавать контент к игре, от персонажей и моделей локаций.

Для Звукорежиссеров

Цель Вертикального среза – получить минимально возможную полноценную версию игры, включающую в себя полностью реализованный основной игровой процесс. При этом высокое качество проработки обязательно нужно воплотить только для тех игровых элементов, которые существенно влияют на восприятие продукта.
